
# Movie App
Movie app made in Flutter with clean architecture using BLoC, Hive and the movie db API.

## Features
- Search feature
- Watchlist feature
- Now playing movies
- Popular movies
- Top rated movies
- Movie details
- Movie cast
- Movie reviews
- Similar movies
- On air tv shows
- Popular tv shows
- Top rated tv shows
- TV show details
- Similar tv shows
- TV show season details
